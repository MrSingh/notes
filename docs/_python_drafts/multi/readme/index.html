Known Limitations

  • Sales volume in HPI lags price data by ~2 months — not joined on same date
  • FRED CPI used for cross-validation only; ONS L55O is primary source
  • FCA firm names fuzzy-matched; match confidence < 0.85 excluded

How to Reproduce

  1. Download CSVs using links in notebooks/Multi_Source_Data_Consolidation.ipynb Section 1
  2. Place files in data/raw/
  3. Run notebook sections 2–4 in order
  4. Output dashboard at data/outputs/dashboard.html

Dependencies

pandas, numpy, plotly, requests, jinja2
