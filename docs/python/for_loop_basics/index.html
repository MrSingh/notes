For Loops Basics

Feb 18, 2021

Create a List of Numbers

nums = [1,2,3,4,5]

Print Numbers in a List Example

for num in nums:
    print(num)

1
2
3
4
5

Continue Example

#continue moves to the next iteration - in the example below when num == 4, 'Found!' will be printed and it will skip print(num) and go to the next iteration of the loop
for num in nums:
    if num == 4:
        print('Found!')
        continue
    print(num)

1
2
3
Found!
5

Break Example

#break, breaks out of the loop
for num in nums:
    if num == 4:
        print('Found!')
        break
    print(num)

1
2
3
Found!

Nested For Loop Example

#loop within a loop
for num in nums:
    for letter in 'abc':
        print(num, letter)
